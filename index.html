How it works
1
Enter a wallet address. A Stellar public key is a unique 56-character ID that identifies an account on the Stellar blockchain.
2
We call the Horizon API. Horizon is Stellar's official HTTP API — it lets apps read data from the Stellar network without needing a full node.
3
Data comes from the ledger. Stellar's blockchain is a distributed ledger replicated across thousands of nodes. Every balance and transaction is stored there permanently.
4
Your dashboard appears. StellAura renders the data in real time — balance, stats, and full transaction history. Completely read-only. No private keys involved.
